## 💻 Cara Menjalankan Project
1. Clone repository ini:
   ```bash
   git clone [https://github.com/fitria-lm/mobile-app-kuliah.git](https://github.com/fitria-lm/mobile-app-kuliah.git)

```

2. Buka folder tugas yang diinginkan (misal `tugas1-calculator`).
3. Jalankan melalui IDE **IntelliJ IDEA** atau **Android Studio**.
4. Masukkan ekspresi matematika dan tekan **Enter**.
